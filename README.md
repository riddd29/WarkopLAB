## 🚀 How To Run?

### Prasyarat
- Java 17 atau lebih tinggi
- Maven
- MySQL (dengan skema dan user yang sesuai)

### Langkah-Langkah

```bash
# Clone repositori
git clone https://github.com/riddd29/WarkopLAB.git
cd warkoplab

# Jalankan aplikasi
./mvnw spring-boot:run
